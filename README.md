# Logo Showcase (Next.js + Three.js)

واجهة تفاعلية لعرض شعارك بطبقات، تأثيرات 3D غير منتهية، شرح الخطوات، ومعرض موك أب.

## المتطلبات
- Node.js 18+

## التشغيل
```bash
npm install
npm run dev
```
ثم افتح `http://localhost:3000`.

## أين أضع ملفاتي؟
- ضع صور طبقات الشعار (PNG/JPG/WebP/SVG بخلفية شفافة إن أمكن) داخل:
  - `public/layers/`
- ضع صور الموك أب داخل:
  - `public/mockups/`

سيقوم النظام بقراءة الملفات تلقائيًا (عبر API) وعرضها بالترتيب الأبجدي. إن كانت أسماء الملفات مرقمة (01, 02, 03…) سيُحافظ على الترتيب الصحيح.

## المحتوى النصي
حرر الملف:
- `content/project.json`
لتحديث العنوان، الوصف، وخطوات الشرح الزمنية.

## المزايا (قيد التنفيذ)
- تأثير خلفية 3D إبداعي غير منتهٍ.
- تكديس طبقات الشعار كمسطحات في مشهد ثلاثي الأبعاد مع انكشاف تدريجي.
- خط زمني يشرح مراحل التنفيذ.
- معرض موك أب متجاوب.
- متوافق مع الجوال.

## البناء والإطلاق
```bash
npm run build
npm start
```
